# 主なビジネス目標

主要なビジネス目標は、組織がデジタルエクスペリエンスイニシアチブを通じて達成しようとする戦略的成果を定義します。 各目標は、Adobe Experience Platformを実装する方法と、それらの成果を実現するアプリケーションを記述した 1 つ以上の [ ユースケースパターン ](/help/blueprints/use-case-patterns/overview.md) にマッピングされています。

次の表を使用して、組織に最も関連性の高いビジネス目標を特定し、リンクに従って、各目標をサポートするユースケースパターン、KPI、実装アプローチを確認します。

## 獲得と成長

次の目標は、リーチの拡大と顧客基盤の拡大に焦点を当てています。

| 目的 | 説明 | 主要な KPI |
| --- | --- | --- |
| [ 新規顧客獲得 ](acquisition-growth/acquire-new-customers.md) | ターゲットを絞った獲得キャンペーン、類似オーディエンス、有料メディア最適化を通じて、顧客基盤を拡大します。 | 新規顧客、顧客獲得コスト、見込み客/リードコンバージョン |
| [ リードジェネレーションの向上 ](acquisition-growth/increase-lead-generation.md) | フォーム、イベント、コンテンツ、マルチチャネルエンゲージメントを通じて、セールスパイプラインでより適格なリードを生成します。 | 見込み客、リードあたりのコスト、リードコンバージョン |
| [Web サイトエンゲージメントの向上 ](acquisition-growth/increase-website-engagement.md) | 関連するエクスペリエンスを通じて、サイトでの時間、セッションあたりのページ数、web コンテンツとのインタラクションを改善します。 | ページ滞在時間、エンゲージメント、コンバージョン率 |

## 収益と収益化

次の目標は、収益の増加と取引価値の向上に焦点を当てています。

| 目的 | 説明 | 主要な KPI |
| --- | --- | --- |
| [ コンバージョン率の向上 ](revenue-monetization/increase-conversion-rates.md) | 購入、サインアップ、フォーム送信などの目的のアクションを完了する訪問者や見込み客の割合を改善します。 | コンバージョン率、リードコンバージョン、リードあたりのコスト |
| [ 収益と売上の増加 ](revenue-monetization/increase-revenue-sales.md) | 最適化されたデジタルチャネル、キャンペーン、カスタマージャーニーを通じて、トップレベルの売上高の増加を推進します。 | — |
| [ クロスセルとアップセルの収益を促進 ](revenue-monetization/drive-cross-sell-upsell-revenue.md) | 行動および購入履歴に基づいて、既存の顧客に補完的でプレミアムな製品またはサービスを宣伝します。 | アップセル/クロスセル %、増分売上高、顧客のライフタイムバリュー |
| [ 顧客ロイヤルティとライフタイムバリューの向上 ](revenue-monetization/increase-customer-loyalty-lifetime-value.md) | ロイヤルティプログラム、報酬、パーソナライズされたエンゲージメントを通じて、顧客関係を深め、長期的な価値を最大化します。 | 顧客のライフタイム値、リテンション、アップセル/クロスセル % |

## コストと効率

以下の目標は、コスト削減とマーケティング効率の向上に焦点を当てています。

| 目的 | 説明 | 主要な KPI |
| --- | --- | --- |
| [ 顧客獲得コストの削減 ](cost-efficiency/reduce-customer-acquisition-cost.md) | ターゲティング効率の向上、獲得キャンペーンからの既存顧客の抑制、メディア支出の最適化を実現します。 | 顧客獲得コスト、リードあたりのコスト、効率 |
| [ マーケティング費用と ROI の最適化 ](cost-efficiency/optimize-marketing-spend-roi.md) | ターゲティング、アトリビューション、オーディエンス抑制、予算配分の改善を通じて、マーケティング投資回収率（ROI）を向上させます。 | コスト削減、顧客獲得コスト、収益増加 |
| [ データ品質とガバナンスの向上 ](cost-efficiency/improve-data-quality-governance.md) | 正確なターゲティング、無駄の削減、信頼性の高い分析のために、クリーンで完全な準拠データを確保します。 | 効率とコスト削減 |
| [ マーケティングテクノロジーの統合と最新化 ](cost-efficiency/consolidate-modernize-marketing-technology.md) | 統合されたスケーラブルなプラットフォームに移行することで、ツールの断片化と技術的負担を軽減します。 | コスト削減、効率性、市場投入までのスピード |

## 顧客体験

以下の目標は、カスタマーライフサイクル全体でより優れたエクスペリエンスを提供することに焦点を当てています。

| 目的 | 説明 | 主要な KPI |
| --- | --- | --- |
| [ パーソナライズされたカスタマーエクスペリエンスの提供 ](customer-experience/deliver-personalized-customer-experiences.md) | コンテンツ、オファーおよびメッセージングを、個々の環境設定、行動およびライフサイクルステージに合わせて調整します。 | エンゲージメント、コンバージョン率、顧客満足度（CSAT） |
| [ 顧客維持の向上 ](customer-experience/improve-customer-retention.md) | 価値主導のエクスペリエンスと継続的な関係育成を通じて、既存の顧客のエンゲージメントと更新を維持します。 | 保持、顧客の生涯価値、エンゲージメント |
| [ 顧客のオンボーディングの改善 ](customer-experience/improve-customer-onboarding.md) | 合理化され、パーソナライズされたウェルカムエクスペリエンスとアクティベーションジャーニーにより、新規顧客の価値実現までの時間を短縮します。 | エンゲージメント、リテンション、コンバージョン率 |
| [ 放棄された買い物かごとジャーニーの復元 ](customer-experience/recover-abandoned-carts-journeys.md) | 購入、申請、登録のフロー中に中断したユーザーを再エンゲージし、パーソナライズされたタイムリーなフォローアップを行います。 | コンバージョン率、増分売上高、エンゲージメント |

## 分析とインサイト

次の目標では、測定、レポート作成、データに基づく意思決定の改善に焦点を当てています。

| 目的 | 説明 | 主要な KPI |
| --- | --- | --- |
| [ 分析とレポートの改善 ](analytics-insights/improve-analytics-reporting.md) | 統合ダッシュボードとセルフサービスツールを通じて、レポート機能を強化し、より迅速で実用的なマーケティングインサイトを得ます。 | 効率と生産性 |
| [ データに基づく意思決定の有効化 ](analytics-insights/enable-data-driven-decision-making.md) | セルフサービス分析、リアルタイムの顧客インサイト、戦略の指針となる AI を活用した予測で、チームを強化します。 | 効率と生産性 |
| [ マーケティングアトリビューションの改善 ](analytics-insights/improve-marketing-attribution.md) | マーケティングタッチポイント、チャネルおよびキャンペーンがコンバージョンや収益の成果に与える影響を正確に測定します。 | 効率性、収益増加 |
